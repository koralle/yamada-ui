## ARIAロールと属性

| コンポーネント | ロールと属性   | 使い方                                                                                                                                                                                             |
| -------------- | -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Avatar`       | `role="img"`   | 画像であることを示します。                                                                                                                                                                         |
|                | `name`         | アバターが表す人物の名前。                                                                                                                                                                         |
|                | `alt`          | アバター画像の代替テキスト。`alt`が設定されていない場合は、`name`の値を設定します。                                                                                                                |
|                | `src`          | アバター画像のパス。                                                                                                                                                                               |
|                | `aria-label`   | `src`が設定されていない、画像の読み込みが完了していない場合は、`name`の値を設定します。`name`が設定されていない場合は`alt`の値を設定し、`alt`が設定されていない場合は`"Avatar Icon"`を設定します。 |
| `AvatarGroup`  | `role="group"` | グループであることを示します。                                                                                                                                                                     |
