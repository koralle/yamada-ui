`Alert`は、アクセシビリティに関して[WAI-ARIA - Alert Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/alert/)に従います。

## ARIAロールと属性

| コンポーネント | ロールと属性   | 使用方法                       |
| -------------- | -------------- | ------------------------------ |
| `Alert`        | `role="alert"` | アラートであることを示します。 |
